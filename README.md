#  Task-6: Password Strength Evaluation & Brute-Force Test

##  Objective
To understand the principles of strong password creation, evaluate password strength using online tools, and simulate a brute-force attack using a custom wordlist.

---

##  Tools Used

- [PasswordMeter](https://passwordmeter.com)
- Bitwarden Password Generator
- **Gobuster** for brute-force on web login
- Custom password list (`pass.txt`)
- TryHackMe vulnerable lab (test target)


## Passwords Tested

| Password                        | Strength     | Notes                        |
|----------------------------------|--------------|------------------------------|
| 12345678                        | Very Weak    | Common and short             |
| password123                     | Weak         | Easily guessed               |
| Welcome@2023                    | Moderate     | Slightly complex             |
| P@ssW0rd!99                     | Strong       | Good mix of characters       |
| Books-Fly-Under-Moon-23         | Very Strong  | Passphrase, long and secure  |


## Key Concepts

- Password Strength
- Brute Force & Dictionary Attacks
- Password Complexity
- Authentication Mechanisms
- Passphrases
- Password Managers
- Secure Password Practices
